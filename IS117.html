Introduction to Web Page Development
Building Web Applications

Introduction to Web Page Development is an information systems course designed to instruct students on:

  • Create a multi-page website using HTML and CSS
  • Use a structed construction process consisting of defining requirements, creating a mockup, and coding the final website.
  • Select appropriate image file format for use online.
  • Use Linux commands to manage files and folders on a web server.
  • Design and create a navigation menu.
  • Define basic terminology used by web page analytics tools such as Google Analytics
  • Explain how web pages are accessed and transmitted across the Internet.
